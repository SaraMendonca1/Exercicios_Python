# Resolução da Lista de Exercícios: Python 3

Este repositório contém as soluções desenvolvidas para a lista de 20 exercícios práticos, abrangendo desde a lógica de programação fundamental até a manipulação avançada de dados e o paradigma de Orientação a Objetos.

---

## Configuração do Ambiente de Desenvolvimento

A execução dos scripts contidos neste repositório requer um ambiente configurado adequadamente. Para orientações detalhadas sobre a preparação do sistema, consulte o guia oficial que acompanha este material: **Instalando_Python_e_VS_Code_no_Windows.pdf**.

### Requisitos de Sistema

* **Linguagem:** Python 3.10 ou superior.
* **Editor:** Visual Studio Code.
* **Dependências Externas:**
* Pandas (Análise de dados).



---

## Índice de Exercícios

Os desafios estão organizados por níveis de complexidade, permitindo uma progressão didática através das funcionalidades da linguagem.

### Nível I: Fundamentos e Lógica Inicial

* **Exercício 1:** Olá Mundo Personalizado – Entrada e saída de dados.
* **Exercício 2:** Calculadora Simples – Operadores aritméticos.
* **Exercício 3:** Verificador de Paridade – Lógica condicional.
* **Exercício 4:** Tabuada Automática – Estruturas de repetição (for loops).
* **Exercício 5:** Média de Notas – Controle de fluxo e tomada de decisão.

### Nível II: Estruturas de Dados e Modularização

* **Exercício 6:** Lista de Compras – Manipulação de listas dinâmicas.
* **Exercício 7:** Contador de Vogais – Processamento de strings e funções.
* **Exercício 8:** Dicionário de Contatos – Mapeamento de dados com chave-valor.
* **Exercício 9:** Fatorial – Implementação de recursividade.
* **Exercício 10:** Conversor de Temperatura – Funções com múltiplos parâmetros.

### Nível III: Manipulação de Arquivos e Segurança

* **Exercício 11:** Leitor de Arquivos – Entrada/Saída (I/O) e tratamento de erros.
* **Exercício 12:** Gerador de Senhas – Uso das bibliotecas random e string.
* **Exercício 13:** Registro de Logs – Manipulação de datas com o módulo datetime.
* **Exercício 14:** Validador de CPF – Implementação de algoritmos de validação.
* **Exercício 15:** Persistência em CSV – Exportação e leitura de dados estruturados.

### Nível IV: Ciência de Dados e Objetos

* **Exercício 16:** Análise de Dados com Pandas – Filtragem e sumarização de DataFrames.
* **Exercício 17:** Sistema Bancário – Abstração, encapsulamento e classes.
* **Exercício 18:** Gráficos Estatísticos – Visualização de dados com Matplotlib.
* **Exercício 19:** Web Scraping – Coleta automatizada de dados via HTTP.
* **Exercício 20:** Jogo da Forca (POO) – Integração de lógica de jogo em arquitetura de classes.

---


## Notas Técnicas

Todo o código foi escrito seguindo as convenções da PEP 8 para garantir a legibilidade e manutenibilidade. Os comentários internos detalham o funcionamento de algoritmos mais complexos, como os validadores e os seletores de dados do Pandas.
