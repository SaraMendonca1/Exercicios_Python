LISTA DE COMPRAS EM PYTHON
Este é um projeto simples em Python que simula uma Lista de Compras via terminal. O objetivo é permitir que o usuário gerencie itens de forma dinâmica, utilizando conceitos fundamentais como listas, loops e estruturas condicionais.

FUNCIONALIDADES
O programa oferece um menu interativo com as seguintes opções:

• Adicionar itens: Permite inserir novos produtos na lista.

• Exibir lista: Mostra todos os itens cadastrados com numeração.

• Excluir item: Remove um produto específico da lista através do nome.

• Sair: Finaliza a execução do programa.

TECNOLOGIAS UTILIZADAS
• Linguagem: Python 3.10+

• Estruturas principais: while, match/case, try/except e manipulação de listas (append, remove).

COMO EXECUTAR
1. Certifique-se de ter o Python instalado em sua máquina.

2. Copie o código para um arquivo chamado main.py.

3. Abra o terminal na pasta onde o arquivo foi salvo.

4. Execute o comando:

python main.py

EXEMPLO DE USO
----- COMPRAS -----
1 - Adicionar itens
2 - Exibir lista
3 - Excluir item
0 - Sair
Digite a opção desejada: 1
Insira o nome do item: Café
