SISTEMA DE AVALIAÇÃO
Este script em Python foi desenvolvido para automatizar o cálculo da média aritmética de um aluno e determinar sua situação acadêmica com base em uma nota de corte.

FUNCIONALIDADES
• Coleta de três notas distintas através da entrada do usuário.

• Cálculo da média aritmética simples.

• Formatação do resultado com duas casas decimais.

• Verificação lógica da situação do aluno (Aprovado ou Reprovado).

TECNOLOGIAS UTILIZADAS
• Python 3

REGRAS DE NEGÓCIO
O programa segue o fluxo de decisão padrão:

• Média >= 7.0: O aluno é considerado Aprovado(a).

• Média < 7.0: O aluno é considerado Reprovado(a).

COMO EXECUTAR
1. Certifique-se de que o Python está instalado em seu sistema.

2. Salve o código em um arquivo chamado main.py.

3. Execute o programa via terminal:

python main.py

EXEMPLO DE SAÍDA
Ao inserir as notas, o sistema processará os dados conforme o exemplo abaixo:

Digite 1ª nota: 8.5

Digite 2ª nota: 7.0

Digite 3ª nota: 6.0

Média: 7.17

Aprovado(a)!
