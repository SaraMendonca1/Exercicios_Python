OLÁ MUNDO PERSONALIZADO
Este é um script simples em Python desenvolvido como um exercício prático para iniciantes. 
O objetivo do programa é interagir com o usuário, capturando seu nome e retornando uma saudação personalizada no console.

FUNCIONALIDADES
• Solicita a entrada de texto do usuário (nome).

• Utiliza f-strings para formatação de mensagens.

• Exibe uma recepção calorosa e personalizada.

TECNOLOGIAS UTILIZADAS
• Python 3 (ou superior)

COMO EXECUTAR O PROGRAMA
1. Certifique-se de ter o Python instalado em sua máquina.

2. Copie o código abaixo para um arquivo chamado main.py:

nome = input("Informe seu nome: ")
print(f"Olá, {nome}! Bem-vindo(a) ao mundo Python!")

3. Abra o terminal ou prompt de comando na pasta do arquivo.

4. Execute o programa com o seguinte comando:

python main.py

EXEMPLO DE USO
Ao rodar o script, o resultado será semelhante a este:

Informe seu nome: Sara
Olá, Sara! Bem-vindo(a) ao mundo Python!
