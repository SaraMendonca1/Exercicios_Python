VERIFICADOR DE PAR OU ÍMPAR
Este é um script simples em Python projetado para determinar se um número inteiro fornecido pelo usuário é par ou ímpar. 
É um exemplo clássico de uso de estruturas condicionais e operadores aritméticos.

FUNCIONALIDADES
• Captura de entrada numérica do usuário.

• Uso do operador de módulo (%) para verificar o resto da divisão.

• Estrutura condicional if/else para direcionar a resposta.

TECNOLOGIAS UTILIZADAS
• Python 3

LÓGICA DE PROGRAMAÇÃO
O programa utiliza o operador % (resto da divisão) por 2:

• Se o resto for 0, o número é classificado como Par.

• Se o resto for diferente de 0, o número é classificado como Ímpar.

COMO EXECUTAR
1. Tenha o Python instalado em seu computador.

2. Salve o código em um arquivo chamado main.py.

3. Execute o script através do terminal:

python main.py

EXEMPLO DE SÁIDA
Ao fornecer um número, o programa retornará uma mensagem como nos exemplos abaixo:

Exemplo 1:

Digite um número: 10

O 10 é par!

Exemplo 2:

Digite um número: 7

O 7 é ímpar!
