CALCULADORA SIMPLES NO PYTHON
Este projeto consiste em uma calculadora interativa executada via terminal. 
O programa permite realizar operações matemáticas básicas de forma contínua até que o usuário decida encerrar a execução.

FUNCIONALIDADES
• Menu interativo para seleção de operações.

• Operações de Soma, Subtração, Multiplicação e Divisão.

• Tratamento de erros para entradas que não sejam números (ValueError).

• Verificação de segurança para evitar a divisão por zero.

• Estrutura de repetição que permite realizar múltiplos cálculos sem reiniciar o programa.

TECNOLOGIAS UTILIZADAS
• Python 3

• Estrutura condicional match-case (disponível a partir do Python 3.10)

• Blocos de tratamento de exceção try-except

COMO EXECUTAR
1. Certifique-se de ter o Python 3.10 ou superior instalado em sua máquina.

2. Salve o código em um arquivo chamado main.py.

3. Abra o terminal na pasta onde o arquivo foi salvo.

4. Execute o comando:

python main.py

ESTRUTURA DO MENU
Ao iniciar, o programa exibirá as seguintes opções:

• 1 - Somar

• 2 - Subtrair

• 3 - Multiplicar

• 4 - Dividir

• 0 - Sair

EXEMPLO DE USO
----- CALCULADORA SIMPLES -----
1 - Somar
2 - Subtrair
3 - Multiplicar
4 - Dividir
0 - Sair
